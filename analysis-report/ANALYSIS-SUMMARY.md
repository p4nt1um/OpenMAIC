## 🔑 关键发现摘要

### 项目定位
OpenMAIC 是一个 **AI 驱动的多智能体交互式课堂平台**，能够将任何主题或文档转化为包含 AI 教师和同学的互动课堂体验。

### 核心技术亮点

1. **无状态服务器架构**
   - 客户端维护所有会话状态
   - 支持水平扩展
   - 提高系统弹性

2. **LangGraph 多智能体编排**
   - 导演-代理模式
   - 支持 28+ 种动作类型
   - 实时讨论和辩论

3. **两阶段生成管道**
   - 大纲生成 → 场景内容生成
   - 支持幻灯片、测验、模拟、PBL

4. **SSE 流式通信**
   - 实时事件推送
   - 心跳机制
   - 指数退避重试

### 安全态势
- **防护**: SSRF 保护、凭证转发防护
- **待改进**: 速率限制、CSP 头部、认证增强
- **安全等级**: 中-高

### 技术栈
- **前端**: Next.js 16.1.2, React 19.2.3, Tailwind CSS 4
- **AI**: LangGraph 1.1.1, LangChain 1.1.16, Vercel AI SDK 6.0.42
- **状态**: Zustand 5.0.10 + Dexie 4.2.1 (IndexedDB)
- **部署**: Docker 多阶段构建
